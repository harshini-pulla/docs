# Category Management Workflow

This workflow explains how admins manage categories, sub-categories, and sub-sub categories inside the RMS.

![Screenshot 2025-11-28 111117.png](/images/Screenshot2025-11-28111117.png)

## Add Category

1. Click **New Category**
2. Enter Category Name
3. Add Description
4. Set Status (Active / Inactive)
5. Save the category

![Screenshot 2025-11-28 111016.png](/images/Screenshot2025-11-28111016.png)

## Edit Category

1. Click **Edit**
2. Update name, description, or status
3. Save changes

![Screenshot 2025-11-28 111038.png](/images/Screenshot2025-11-28111038.png)

## View Category

- Shows details of category
- Lists its sub categories and sub-sub categories

![Screenshot 2025-11-28 111026.png](/images/Screenshot2025-11-28111026.png)

## Delete Category

1. Click **Delete**
2. Confirm deletion
3. Category is removed

![Screenshot 2025-11-28 111055.png](/images/Screenshot2025-11-28111055.png)

## Add Sub Category

1. Click **Add Sub Category**
2. Enter name, description, total experience
3. Save
4. Repeat to add unlimited sub categories

## Add Sub Sub Category

1. Inside each Sub Category, click **Add Sub Sub Category**
2. Enter name, description, total experience
3. Select Target Companies
4. Save
5. Add unlimited sub-sub categories as needed

## Summary

Admins can create and maintain a complete multi-level taxonomy of:

- Categories
- Sub Categories
- Sub Sub Categories

used across JD Creation, Profile Articulation and Candidate Profiles.
